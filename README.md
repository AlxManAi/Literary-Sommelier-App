# Literary Sommelier | Литературный Сомелье

**Literary Sommelier** — это умный ИИ-ассистент, который подбирает идеальную книгу под ваше настроение. Используя мощь **Google Gemini**, приложение анализирует ваши эмоции и предпочтения, предлагая персонализированные литературные рекомендации с уникальной атмосферой.

## Возможности

*   **Подбор по настроению**: Расскажите, что вы чувствуете («хочу приключений», «грустно», «мечтательно»), и сомелье подберет книгу, которая идеально срезонирует с вами.
*   **Голосовое общение**: Общайтесь с сомелье голосом в реальном времени.
*   **Визуальная атмосфера**: ИИ генерирует уникальные изображения, передающие настроение рекомендованной книги.
*   **Поиск книг**: Автоматический поиск ссылок на книги в популярных онлайн-магазинах/библиотеках.
*   **История**: Сохранение истории рекомендаций, чтобы вы не потеряли интересные находки.

## Технологический стек

*   **Frontend**: React 19, Vite, TypeScript
*   **AI**: Google GenAI SDK (Gemini Modules: Text, Image, Audio)
*   **Styling**: Tailwind CSS
*   **Tooling**: ESLint, PostCSS

## Как запустить локально

1.  **Клонируйте репозиторий**:
    ```bash
    git clone https://github.com/AlxManAi/Literary-Sommelier-App.git
    cd Literary-Sommelier-App
    ```

2.  **Установите зависимости**:
    ```bash
    npm install
    ```

3.  **Настройте API ключ**:
    Создайте файл `.env.local` в корне проекта и добавьте ваш ключ от Google Gemini:
    ```env
    GEMINI_API_KEY=ваш_ключ_здесь
    ```
    > Получить ключ можно в [Google AI Studio](https://aistudio.google.com/).

4.  **Запустите проект**:
    ```bash
    npm run dev
    ```
    Приложение будет доступно по адресу `http://localhost:3000`.
